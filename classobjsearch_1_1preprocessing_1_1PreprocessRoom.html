void objsearch::preprocessing::PreprocessRoom::transformAndRemoveFloorCeiling ( pcl::PointCloud< pcl::PointXYZRGB >::Ptr &  cloud,
const tf::StampedTransform &  roomTransform 
)

The cloud we receive is translated and rotated relative to the global frame. It is possible to extract this from the gathered data using the XML provided with each room indicating the rotation of intermediate clouds. We use the rotation of the first intermediate cloud to translate and rotate the merged cloud into its actual position, and then threshold the z-axis values according to the height of the floor and ceiling (plus some offset) to remove points which make up those structures.

Parameters
cloudCloud to transform and remove the floor and ceiling from.
roomTransformThe transform needed to put the cloud into its position relative to the global reference frame. Ideally extracted from the XML data for the room and its intermediate clouds.
