Annotate points in a cloud loaded from a certain directory based on the annotations. Points in the given cloud will be compared to the annotated objects, and labelled with the label of the nearest object, but only if the point is within a euclidean distance of maxDist of the object. Uses nearest neighbour search for each point in the query cloud.

Parameters
dirThe top level room directory containing the cloud of interest.
cloudThe cloud containing points to annotate.
indicesThis vector will be populated with the indices of the points which have been labelled
labelsWill be populated with the labels of the points
distancesWill be populated with the minimum distance of the point to its object
maxDistThe maximum distance from an object a point can be to still be considered part of the object
